# easy_list

Minimalistische To‑Do‑Liste als Vue 3 + TypeScript App.

## Features
- Einträge hinzufügen per Button oder Enter
- Erledigte Einträge werden nach unten sortiert
- Erledigte Einträge entfernen
- LocalStorage Persistenz (`easy-list-entry`)
- Sanitizing gegen `<script>`

## Installation
```bash
npm install
```

## Entwicklung starten
```bash
npm run dev
```

## Production Build
```bash
npm run build
```

## Vorschau des Builds
```bash
npm run preview
```
